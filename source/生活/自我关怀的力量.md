# 自我关怀的力量
## 为什么自我关怀
为什么我们不愿承认脱离常规之后的粗鲁和不耐烦呢？那是因为把自己的缺点和不足投射到别人身上会让自我感觉良好。每个人都因说错话或做错事而责怪他人，以此来证明自己做法的合理性。但与此同时，我们的内心又深知，究竟我们还要在这样的事情上浪费多少时间？若我们能平心静气、坦诚相待，岂不是更好？

但是，说起来容易做起来难。如果我们不能透彻地看待自己，也就不可能留意到自己身上引发问题的点，也不可能留意到妨碍自身潜力发挥的方面。如果我们无法承认自已的缺点，又怎能一步步成长呢？忽视自己的缺点，认为自己的问题和困难都是他人的责任，可以暂时让我们感觉良好；但是长此以往却对我们有百害而无一利，让我们陷入停滞不前和冲突的恶性循环中。
### 揭秘自我关怀
#### 自我评判的代价
不断满足积极自我评价的需要似乎有点儿像饮酒，虽然甘醇，却留有后患。而就在隐患暴露之余，我们也恍乌——不管我们是否情愿，不能总把问题归咎于他人。我们不能总是感觉非同一般、出类拔萃。

那么解决之道是什么呢？ 一言以蔽之：**停止评判和评价自己。** 不再给自己贴上“好”或“坏”的标签，以开放的心态接纳自己。友善、关切和体恤地对待自己，就像对待朋友甚至陌生人一样。我们从没有像虐待自己一样虐待过任何人。

从佛教的观点来看，只有学会了关心自己，才能真正地关心他人。如果你继续评判和批评自己，却努力善待他人，就会认为地划分出一条边界，从而导致隔离和鼓励感。

尽管自我批判过得到社会的赞成，却丝毫没有帮助，它只能让事情变得更糟。若自我批评了，我反而会感到不称职和不安全，并且向周围的朋友抱怨自己的挫败感，甚至于不敢承认自己的错误，因为我害怕随之而来的对自己的痛恨。

#### 关怀他人和自己
关怀是对痛苦遭遇的察觉和洞悉。它还是对身陷苦境的人的善意，所以关怀的欲望——排忧解难——就应运而生。最后，关怀还认可我们共同的人类境遇——它布满瑕疵，不堪一击。

自我关怀，按照定义，与关怀他人包含了同样的品质。它需要我们停下脚步，认可自身的遭遇。如果我们甚至都不承认它的存在，便不可能被自身的痛苦所打动。当然，有时我们会被痛苦所蒙蔽，对任何事都视而不见。

每个人都会时不时犯错，人生大抵如此。为什么谨言慎行以后，就指望所有事情都变好呢？

活在推崇独立和个人成就的文化氛围中具有诸多的负面效应，其中之一就是如果我们无法不断地达到自己的目标，剩下的就只有自责而已。而如果我们犯了过错，就意味着不配再得到关怀，这样对吗？其实每个人都值得关怀才是真理所在。作为生活于这个星球上具有意识的人类，其本身就具有内在的价值，同样也值得关爱。

自我关怀是通完幸福和惬意生活的康庄达到。给予自己无条件的关切和安慰，同时感受人们所有的体验，尽管困难依旧，我们却避免了恐惧、否定和疏离的袭扰。同时，自我关怀还滋养了幸福和乐观的积极心态。正是因为有自我关怀的呵护，我们才能健康成长，也更懂得即使身处艰难时刻也能感恩生活的多姿多彩，一路风雨无阻。当焦灼得到关怀的抚慰，是非也变得清晰可辨，它指引着我们奔向快乐的源泉。

### 终结疯狂的行为
不健康的心理习惯：我们会为了让自我感觉良好，扭曲对整个世界的感知。就好像只有不断地矫饰自我形象才能使之符合我们的喜好，甚至会不惜扭曲现实。不过当理想受挫的时候，我们又会无情地批评自己，程度只严厉又走向另一个极端。

我们为何在让自己心里更好过而扭曲事实而无情的自我批评之间摇摆呢？这是因为我们渴望安全。不管是作为一个物种，还是单一的个体，人类的发展都是基于基本的生物本能。

乌比冈湖效应：人们自认为在许多积极的人格特质上都优于他人的普遍倾向。我们不仅仅把自己看做是更好的，也会把其他人看成是更差的。

此种做法有许多严重的后患，当我们总是看到他人身上的不足时，感知不免收到消极阴暗面的遮蔽。我们的所思所想也会变得卑劣，如此一来，我们的心理世界也会浑浊一片。下行社会比较实际上对我们有百害而无一利。把他人踩在脚下、让自己高高在上实际上害人害己，它让我们变得孤立和隔绝，而这恰恰是我们所极力回避的。

或许更让人匪夷所思的是，为何自我批评的倾向会如此之强。自我批评的语言就像一把尖刀割在身上。

大多数自我批评的想法都采用内心对话的形式，即不断地品评自己的经历。虽然内心对话非常残酷严厉，但由于不存在社会舆论的谴责，所以我们与自己的对话通常以极端残忍的方式进行。

#### 父母的角色
家人是与我们的生存最息息相关的社会群体。儿童本能的信任父母所提供的食物、环境、住处，相信父母会帮助应对可怕的挑战，使儿童们远离伤害。然而，许多父母并没有给儿童温暖与支持，而是通过不断批评试图控制孩子们。

研究显示，由高度批评式的父母所抚养的孩子长大后更容易批评自己。高度批评式的父母会惩罚孩子们不可取的行为，奖励那些可取的行为，不过这会导致孩子的恐惧和不信任，导致孩子们了解排斥和拒绝将不可避免。

儿童对言语暴力的自然反应就是自我保护，而有时最有效的防御手段就是“无短可揭”，这意味着自我批评。

#### 文化的角色
自我批评和无价值感的倾向还可以追溯到更大的文化框架中。在许多地方，自我批评都被认为是一种动机驱动力。例如儒家文化的理想是修身克己，满足他人的需要，而不是自己的。而佛教文化则更可能自我关怀。

#### 控制欲和不安
很容易发现，严厉的自我批评通常隐含着强烈的控制欲。

这通常意味着，我们要为自己的失败负责，其意义在于失败是个雷区，而不完美也应该是能够避免的。当我们评判自我时，我们通常身兼批评者和被批评者的双重角色。进行自我批判，能够给我们以力量感。当我们进行自我批判的时候，也会拥有对那个失败的自我的优越感。这仅仅是一种“荒谬的特权”。

通过设置不切实际的过高标准，以及失败之后的沮丧之情，我们可以巧妙地强化与高标准相联系的卓越感。

当自我惩罚带着一丝幽默，变得取悦他人的时候，就如常言道“与人同笑总好过被人嘲笑”。不过，在健康的自嘲式幽默和不健康的自贬式幽默之间还有差异。第一个是个体足够自信，敢于自我嘲笑；第二个则在于根深蒂固的有关个人价值的不安全感。

另外，自我批评还常常导致不信任他人，过度敏感，更有害的还会损害亲密关系。

**“自我验证理论”：** 渴望自我观得到验证，因为它有助于提供生活中的稳定感。

自我批评不仅导致不安全感，还导致不胜任感以及自卑感。更多的，自我批评者更可能选择自杀。羞耻感和无意义感让人们把自己看的一无是处。

#### 出路
消除自我批评的最佳方式是理解它，对之抱以关心，并代之以更为友善的反应。

## 自我关怀的核心部分
1. 善待自己，以友爱的方式理解我们自己，而不是严厉地批评和指责；
2. 它需要认识到共通人性，感受到与他人在生命体验上的契合，而不是被自己的痛苦所鼓励和隔离；
3. 静观当下，对我们的体验持以平衡的觉察，而不是忽视或者夸大我们的痛苦。

### 善待自己的途径
1. 停止自我批判：  
   停止对自我的不断批判，不再贬低大多数人都认为很正常的内心评论。理解自己的瑕疵和失败，不再一味地加以谴责。我们要看到自己正在动用无情的自我批判伤害自身。善待自己意味着内心交战的终止。
2. 积极主动地安慰自己：  
   像好朋友一样安慰自己，这意味着我们可以被自己的痛苦所打动，向自己传递温暖、友爱和关怀的和平之举。
3. 关爱和营养都是强而有力的生存需要；

我们不应该完全依赖他人，尤其是成年以后，因为其他人可能因为某些原因离开我们，情侣关系可能结束，心理治疗可能无法偿付，我们依赖的人本身可能自顾不暇——疾病、抑郁、工作压力等不仅牵绊着我们自身，也牵绊着他人，从而在我们需要他们的时候，无法抽身。幸运的是，我们并非只能依赖他人来改变我们的自我观。若我们一致性地基于自己关爱和理解，也可以感受到关爱和接纳的价值。若能基于自己以关怀和支持，也会习得信任，助力就在身边。**当我们蜷伏在善待自己的温暖怀抱中时，我们切实的觉得安全无虞。**

当我们对自己体验到温暖和亲切之感时，我们也随之转变着我们的身心。再不用体验忧心和焦虑，代之而来的是镇静、满足、信任与安全。善待自己能让我们面对痛苦体验时原理恐惧之地，拥抱安全。一旦我们抛却了不安全感，就会信心十足地追逐自己的梦想啦。

练习一：拥抱练习
在你感觉糟糕，想要平复和安慰自己时，简单易行的方式是给自己一个温柔的荣宝。

### 我们同舟共济
自我关怀的第二基本元素是认识到共有的人类体验，承认人类的生命息息相关，也正是这一点把自我关怀与自我接纳与自我怜悯区分开来。尽管自我接纳和自我怜悯非常重要，但是本身是不完整的。它让我们远离了一样关键元素——他人。关怀内在地包含着关系属性。自我关怀尊重事实，即所有人都容易犯错误。

在我们理解共通人性时，须记得无力感和失望感人人都有。这也是自我关怀与自我怜悯的区别之处。自我怜悯只会说“可怜的我”，而自我关怀没有忘记每个人都会有痛苦遭遇，我们可以聊以自慰，毕竟我们只是人而已。

在注意到自己生活的不如意时，我们通常会变得羞愤难当。我们会因自己的无能——无法得到想要得到的东西，无法称为想要成为的人——而心生无力感和挫败感。我们抱怨现状，只固守于主观的狭窄视角。实际上，我们命已同舟。

遗憾的是，大多数人没有留意到自己与他人的共通之处，尤其是当失败感扑面而来的时候。越是失败的时候，人们越容易感到孤立和隔离。同时，视角也会变得狭隘，因为我们只关注自身，而忽略了更广泛的人类状况。

心理健康问题的主要原因在于**缺少归属感** ，总觉得我们与其他人割断了联系。

孤独源于无归属感，不管身边是否有人。

**在社会上还容易有比较的问题。** 社会比较最让人伤神的后果之一就是与成功人士之间的距离让我们难过不已，而这种现象极具普遍性。

**我们的心理通常还会自欺欺人。** 认为我们能够也应该比现在更好。对于某些人来说，不完美尤其难以接受。完美主义被界定为强迫自己达成或实现个人目标，绝不允许理想落空。完美主义者为了让事物恰如其分，体验着极大的压力和焦虑。完美主义的积极方面则取决于我们尽全力的坚定性。追求成功和高标准可以是健康有益的特质。

### 秉持静观
自我关怀的第三个关键元素为静观。**静观即对此刻发生的事情保持清醒和非评判性的接纳。** 换言之，直面现实，如实地看待事物本身，不增，不减，才能对当前的境遇抱以最大的——亦最有效的——关怀。

#### 停下脚步觉知痛苦
对自己抱以关怀，首先必须认识到我们正在经受痛苦。若无法感知，便无法治愈。

我们肯定都体验过未能实现理想的痛苦，彼时的心思往往聚焦于失败本身，而不是失败所引起的痛苦。这是差别的关键所在。若只看到对自己的不满意，我们的注意力将会完全被缺陷所占据，也就难以察觉到不完美带来的苦楚，更遑论对之抱以关怀之心。

我们所忽视的不仅是由个人不足之感所引起的痛苦。即使没有个人的责任，若周遭的环境产生变数，我们仍会做出鲁莽的行为。大多数人即使没有因为现状而责备自己，仍倾向于直接解决问题。我们可能会花费大量的时间和经历来应对危机。尽管所做的都有必要，但认识到这些事耗费的心力也非常重要。

我们需要停下脚步，喘息，承认当前的困难，以及对我们的痛苦值得抱以亲切友善的反应。若非如此，我们的痛苦将不会被察觉，压迫感和忧虑会越来越大。由于我们耗费了全部的精力修补外在的问题，而不记得唤醒内心，所以我们将会遭受倦怠、力竭和淹没感的风险。

#### 逃离痛苦体验
压抑或爆发情绪是大多数人都曾体验到的。此过程被本书称为“过度认同”。我们的自我感被裹挟于情绪反应中，以致整个现实都被它们吞噬。我们迷失其中，不再后退一步客观地观察所发生的事。就像我们的所思所感是对现实的直接知觉，已然忘却了我么还有回旋的空间。

过度认同还存在另一个原因，过度反应。此类情绪性过度反应的驱动力则是回避把自己看成“坏”的或有缺陷的。容易过度放大自身的缺陷，却忘了，这些缺陷可能是我们臆想出来的，实际上并没有那么糟糕。

这就是静观的好处之一。静观带我们回到当前，给我们平衡的觉察，也由此构成自我关怀的基础。像一弯澄澈、静谧的清泉，不泛涟漪，静观完美地映照着所发生的事情，不偏不倚。它让我们不再迷失于个人的肥皂剧情中，而是从更大的视角审视我们的处境，帮助我们确信不必再受此困境。

#### 沐浴意识之光
理解静观的关键在于把觉察本身从觉察内容中区分出来。各种各样的事物都出现在我们的觉察之中——躯体感觉、视觉、听觉、嗅觉、味觉、情绪和思维。来了又去，觉察的内容总是在变。即使保持完全的静止，我们还是要喘息、心跳、眨眼，还有声音的起伏。若觉察的内容不再变化，只能是我们已死。按照定义，生命包含着转换与变化。

我们清醒意识之中唯一的静止且恒常之物，只有觉察本身。

```
练习留意：不管什么时候，只要某个特定的思维、情绪或者感觉形成了，我们就要柔和地做个心理记录。
即使忘记了修行，也只要简单地记录“迷失”。
这种训练可以使得我们的大脑更为专注，时时刻刻都对发生于自己的事情保持更好的觉察。使我们更好的投身于当下，给我们应付压力情景所必需的心理视角。
```

#### 反应而非反抗
不再执着于每一个转瞬即逝的想法或情感。冷眼旁观想法和情感的兴亡交替，从而能决定哪些值得注意，哪些不值得。我们可以质疑自己知觉的准确性，询问我们的思想和情感是否值得认真对待。

静观赐予我们最宝贵的财富：就是让我们有机会做出反应而不是简单的反抗。

**折磨=痛苦×对抗** ：情绪上的折磨由对事物的欲望而起，而非事物本身。越是对抗所发生的事情，越是痛苦难熬。痛苦类似气体，如果你任由它在那儿，它会自行消散。答案是如果你对抗和抵制痛苦，把它逼到一个局促空间里，压力则会越来越大，直到爆炸。

#### 理解无能为力
如果能够静观地理解当前时刻，你可以更明智地考虑在下一时刻你想要什么。

静观让我们思考采取什么主动的举措才能改善局面，同时也让我们意识到事物什么时候不能改变，只能接受。

宁静：愿上帝赐我平静的心，接受我不能改变的事情；愿上帝赐我勇气，改变我能改变的事情；愿上帝赐我智慧，明白两者之间的差异。

尽管有些让人膛目结舌，但我们几乎无力改变头脑中的事件。在意识觉察域中到底会生成什么是个迷。思维和情绪往往不请自来，甚至流连忘返。

思维和情绪的产生是基于我们过去的经验和联想、生理构造、激素循环、身体状况、文化学习、先前的思维和情绪以及各种其他因素。我们无法控制穿越意识大门的思维和情绪，但我们却可以改变解读它们的方式。

静观的馈赠是让你接纳当前时刻，更为明智清醒地改造未来时刻。你不仅减少了自己的痛苦煎熬，还能够对下一步该怎么走路做出更好的选择。我们要善待自己的情感，尤其是困难时刻。

自我关怀的是三个入口：
1. 给予自己友善与关切；
2. 提醒自己遭遇困难是人类体验共有的一部分；
3. 对思维和情绪秉持静观。

## 自我关怀的益处
### 情感弹性
自我关怀是对抗焦虑和抑郁的主要保护因素。

与消极信息相比，我们的大脑会把更少的时间和注意力分配给积极信息。

一旦我们聚焦在消极想法上，消极想法就会翻来覆去的播放——“穷思竭虑”。

自我关怀可以缩短消极情绪的持续时间，但并不是以让人厌恶的方式赶走。如果我们强烈的想要抑制住某种想法，可能导致那种想法更大概率会出现。所以，不妨去体验那些困难情绪。

```
这是一个苦难的时刻。
苦难是生活的一部分。
这一时刻，我应该对自己好一些。
我需要给自己一些光环。
```

### 自我关怀与情绪智力
情绪智力：监控自己情绪的能力以及巧妙运用这些信息来指导思维和行动的能力。能够意识到自己的情绪，而不被这些情绪所胁迫。

```
关怀意象：
1. 以舒服的姿势坐在一个安静的地方；
2. 创造一个心目中理想的关怀和慈悲的形象；
3. 如果郑遭受痛苦，想想那些可以安抚你的智慧的、充满关怀的事物；
4. 释放关怀意象，深呼吸并安静的坐着，享受舒适感。
```
